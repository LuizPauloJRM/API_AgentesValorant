Sobre a Valorant Agents

Esta página foi criada como um projeto de estudo para mostrar todos os agentes de Valorant. Aqui você pode navegar pelos agentes, ver suas habilidades e aprender um pouco mais sobre cada um.

O projeto foi desenvolvido utilizando HTML, CSS, Bootstrap e JavaScript puro e consome a API oficial de Valorant.

Desenvolvedor: Luiz Paulo Medeiros

Matricula: 202310962

Documentação do Projeto

Funcionalidades

  • Lista de agentes jogáveis com cards interativos.
  • Modal com informações detalhadas (nome, função, habilidades, descrição, imagens).
  • Barra de pesquisa interativa de armas com modal detalhado.
  • Consumo de API oficial sem necessidade de backend.
  • Navegação entre páginas (Início e Sobre).

Como funciona a API

O projeto consome a API do Valorant via fetch e async/await no JavaScript. Cada agente possui dados como nome, descrição, função, habilidades e imagens. 


async function carregarAgentes() {
    const resposta = await fetch("https://valorant-api.com/v1/agents?isPlayableCharacter=true&language=pt-BR");
    const dados = await resposta.json();
    const agentes = dados.data;
}

As armas também podem ser pesquisadas em tempo real e exibidas em cards com modal de detalhes.
